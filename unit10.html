    Database

    Databases teach us how to use and process data, what it is for and when we should use it. In this unit we are creating a database using the famous Microsoft Office program (Access) a list of at least 50 different clients for a doctor's office that was still registering its patients with sheets of paper. Access is a very powerful program but I believe we will implement SQL soon. In this unit I am learning the technical terms to use and create a database, the Query I forms, I tabs and so on. All these interlinked operations make it possible to create an efficient structure for customers who need a solid database. Clearly, here too there are the pros and cons.
