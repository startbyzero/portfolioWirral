    Digital Portfolio

    The unit is about creating your own digital CV to provide examples of your work and to give future jobs something more to work on, in short, something that demonstrates my skills at whatever level they are. First of all we draw the layout in PowerPoint including all the tabs and information then explain how the whole layout, that is the buttons, the graphics, the fonts used and so on. Then also make a second copy if the client requests another work option. I made the web wallet and uploaded it to the web using GIT HUB. I prefer to create and program the website from scratch, I don't like drag and drop like WordPress and others.
